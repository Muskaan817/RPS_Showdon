# Rock Paper Scissors Game 🎮

A simple Rock Paper Scissors game built using React where users play against the computer with live results and score tracking.

## 🚀 Features
- Interactive UI
- Real-time result display
- Score tracking
- Built with React (Hooks & Components)

## 🛠️ Tech Stack
- React
- JavaScript
- CSS

## ▶️ How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
